　 彼 は 老いる て いる た 。 小さな 船 で メキシコ 湾流 に 漕ぐ 出す 、 独り で 漁 を する て いる た 。 一 匹 も 釣れる ない 日 が 、 既に 八 四 日 も 続く て いる た 。 最初 の 四 〇 日 は 少年 と 一緒 だ た 。 しかし 、 獲物 の 無い まま に 四 〇 日 が 過ぎる と 、 少年 に 両親 が 告げる た 。 あの 老人 は もう 完全 に 「 サラオ 」 だ ん だ よ 、 と 。 サラオ と は 、 すっかり 運 に 見放す れる た という こと だ 。 少年 は 両親 の 言いつける 通り に 別 の ボート に 乗り換える 、 一 週間 で 三 匹 も 立派 だ 魚 を 釣り上げる た 。 老人 が 毎日 空っぽ の 船 で 帰る て くる の を 見る たび に 、 少年 の 心 は 痛む だ 。 彼 は いつも 老人 を 迎える に 行く て 、 巻く た ロープ 、 手鉤 《 ギャフ 》 、 銛 《 もる 》 、 帆 を 巻く つける た マスト など を 運ぶ 手伝い を する の だ た 。 粉 袋 で 継ぎ あて する れる た 帆 は 、 巻き上げる られる て 、 永遠 の 敗北 を 示す 旗印 の よう に 見える た 。
　 老人 は 細い やつれる 、 首筋 に は 深い 皺 が 刻む れる て いる た 。 頬 に は 、 熱帯 の 海 に 反射 する た 日光 によって 、 まるで 皮膚 癌 の よう だ 褐色 の しみ が できる て いる た 。 しみ は 顔 の 両側 に 首 近い まで 連なる て いる 。 両手 に は 、 大きな 魚 の 食らいつく た ロープ を 制する 時 に できる た 、 深い 傷痕 が いくつ も ある た 。 しかし 傷痕 は 新しい もの だ は ない 。 魚 など いる ない 砂漠 、 風 に 侵食 する れる た 砂漠 の よう に 、 古い 傷痕 だ た 。
　 彼 に関して は 何もかも が 古い た 。 ただ 、 その 両 眼 を 除く て は 。 彼 の 眼 は 、 海 と 同じ 色 に 輝く 、 喜び と 不屈 の 光 を たたえる て いる た 。
「 サンチャゴ 」 少年 は 、 船 を 着ける た 岸 の 斜面 を のぼる ながら 老人 に 呼びかける た 。 「 また 一緒 に 行く たい な 。 お金 も 多少 貯まる た し 」
　 老人 は 少年 に 漁 を 教える て くる た 。 少年 は 彼 を 慕う て いる た 。
「 だめ だ 」 老人 は 言う た 。 「 お前 の 船 は ついてる 。 仲間 を 変える ない ほう が いい 」
「 でも 僕ら は 前 に 、 八 七 日 も 不漁 だ た 後 で 、 三 週間 毎日 大物 を 釣る た こと が ある た じゃ ない か 」
「 ある た な 」 老人 は 言う た 。 「 分かる てる さ 。 お前 が 船 を 変える た の は 、 俺 の 腕 を 疑う た から じゃ ない 」
「 親父 だ よ 、 船 を 変える させる た の は 。 僕 は 子供 だ から 、 従う しか ない ん だ 」
「 分かる てる 」 老人 は 言う た 。 「 当然 の こと だ 」
「 親父 に は 、 信じる って こと が できる ない ん だ よ 」
「 そう だ な 」 老人 は 言う た 。 「 でも 俺 たち に は できる 。 そう だ ？ 」
「 うん 」 少年 は 言う た 。 「 テラス で ビール を おごる せる て よ 。 道具 は その後 で 運ぶ う 」
「 いい とも 」 老人 は 応じる た 。 「 漁師 仲間 として 、 頂く う 」
　 二 人 は テラス の 店内 で 腰 を おろす た 。 多く の 漁師 が 老人 を からかう た が 、 彼 は 怒る ない た 。 年配 の 漁師 たち の 中 に は 、 彼 を 見る て 悲しむ 者 も いる た 。 しかし 彼ら は それ を 表 に は 出す ぬ 、 潮 の 流れ とか 、 釣り場 の 水深 とか 、 良い 天気 が 続く て いる とか 、 今日 は 何 を 見る た とか 、 そういう こと を 穏やか に 話す の だ た 。
　 その 日 収穫 の ある た 漁師 たち は とっくに 戻る て いる て 、 カジキ の 処理 も 済ませる て いる た 。 彼ら は 、 二 枚 の 板 いっぱい に カジキ の 身 を 並べる 、 二 人 で 板 の 両端 を 持つ て よろめく ながら 倉庫 へ と 運ぶ だ 。 カジキ を ハバナ の 市場 に 運ぶ 冷蔵 トラック が 来る の を 、 そこで 待つ の だ 。
　 サメ を 獲る く た 漁師 たち は 、 入り江 の 反対 側 に ある サメ 処理 工場 に 獲物 を 運ぶ だ 。 サメ は 滑車 で 吊り上げる られる 、 肝臓 を 取り除く れる 、 ひれ を 切り落とす れる 、 皮 を 剥がれる 、 肉 は 細い 切る れる て 塩漬け に する れる 。
　 風 が 東 から 吹く 時 に は 、 この 工場 の 臭い が 港 を 越える て 漂う て くる た 。 しかし 今日 は ほんの わずか だ 臭い しか 感じる られる ない 。 風 は 北 向き に 変わる 、 それ も やむ で しまう た から だ 。 テラス は 心地よい 陽 に 照らす れる て いる た 。
「 サンチャゴ 」 少年 は 呼びかける た 。
「 ああ 」 老人 は 応える た 。 彼 は グラス を 持つ た まま 、 ずっと 昔 の こと を 考える て いる た 。
「 明日 使う イワシ を 獲る って 来る う か ？ 」
「 いや 。 野球 でも する て 来る と いい 。 俺 は まだ 漕げる し 、 投網 は ロヘリオ が やる だ う 」
「 僕 が 行く たい ん だ よ 。 一緒 に 漁 に 行ける ない だ 、 何 か 役に立つ たい ん だ 」
「 ビール を くれる た じゃ ない か 」 老人 は 言う た 。 「 お前 は もう 一 人前 だ 」
「 初めて 僕 を 船 に 乗せる て くれる た の は 、 何 歳 の とき だ た か な 」
「 五 歳 だ た な 。 釣り上げる た 魚 に 殺す れる そう に なる た ん だ 。 ひどい 活きる の いい 奴 で 、 危うい 船 まで 粉々 に なる ところ だ た 。 覚える てる か ？ 」
「 覚える てる よ 。 尻尾 で バタバタ 跳ねる 回る て 、 船 梁 《 ふる な ばり 》 を ぶち壊す た ん だ 。 棍棒 で ぶん殴る た 時 の 音 まで 覚える てる 。 僕 は サンチャゴ に 舳先 《 へ さき 》 の ほう へ 突き飛ばす れる て 、 濡れる た ロープ の そば で 、 船 全体 が 震える の を 感じる てる た ん だ よ 。 サンチャゴ が 丸太 を ぶつ 割る みたい に 魚 を 棍棒 で 叩く て 、 すごい 音 が する た 。 そこら が 甘ったるい 血 の 臭い で いっぱい に なる た ん だ 。 」
「 本当に 覚える てる の か ？ 　 俺 が する た 話 を 覚える てる だけ じゃ ない の か ？ 」
「 全部 覚える てる よ 。 初めて の 時 から 全部 」
　 老人 は 少年 を 見つめる た 。 老人 の 顔 は 日 に 焼ける 、 その 眼差し は 信頼 と 愛情 に 満ちる て いる た 。
「 お前 が もし 俺 の 子 だ 、 連れる て 行く て 、 いち か ば ちかい 勝負 する ん だ が な 」 彼 は 言う た 。 「 でも お前 は 、 お前 の 親父 と お袋 の 子 だ 。 しか も ついてる 船 に 乗る てる 」
「 イワシ を 獲る って 来る て も いい ？ 　 餌 に する 魚 も 、 四 匹 は 用意 できる よ 」
「 今日 の 残り が ある さ 。 塩 を かける て 箱 に しまう て ある 」
「 新しい の を 四 匹 持つ て 来る よ 」
「 一 匹 だ 」 老人 は 言う た 。 彼 に は 希望 と 自信 が ある 。 それ は 今 、 新しい 風 の よう に 彼 の 中 で 強い なる つつ ある た 。
「 二 匹 だ よ 」 少年 は 言う た 。
「 二 匹 か 」 老人 は うなずく た 。 「 盗む だ もの じゃ ない だ う な ？ 」
「 盗む こと だって できる た けど 」 少年 は 言う た 。 「 買う た ん だ よ 」
「 悪い な 」 老人 は 言う た 。 彼 は 単純 だ た から 、 自分 が いる つ から これ ほど 低 姿勢 だ 人間 に なる た の か など と は 考える ない た 。 自分 が 低 姿勢 に なる た と 自覚 する て は いる た けれど 、 それ が 不名誉 だ こと だ も 真 の 誇り を 損なう もの だ も ない という こと も 分かる て いる た 。
「 この 調子 だ と 、 明日 は いい 天気 に なる そう だ 」 彼 は 言う た 。
「 どこ まで 行く の ？ 」 少年 が 尋ねる た 。
「 ずっと 遠い まで だ 。 風 が 変わる た 戻る 。 明るい なる 前 に 沖 に 出る られる と いい ん だ が 」
「 僕 も 親方 に 、 沖 まで 出る よう に 頼む で みる よ 」 少年 は 言う た 。 「 そう する ば 、 サンチャゴ が すごい 大物 を 引っかける た 時 、 みんな で 手助け に 行ける から ね 」
「 あいつ は 遠出 《 とう で 》 する た がる ない だ う 」
「 そう だ ん だ よ 」 少年 は 言う た 。 「 でも 、 親方 に は 見える ない もの を 僕 が 見つける られる から ね 、 鳥 が 獲物 を 探す てる ところ と か 。 それで シイラ の 後 を 追っかける させる て 、 遠出 する せる て やる ん だ 」
「 あいつ の 眼 は そんなに 悪い の か ？ 」
「 ほとんど 見える てる ない よ 」
「 妙 だ な 」 老人 は 言う た 。 「 あいつ は 海 亀 獲る り は 一 度 も やる ない た ん だ 。 あれ を やる と 眼 に 悪い ん だ が 」
「 でも 、 サンチャゴ は モスキート 海岸 で 何 年 も 海 亀 獲る り を やる てる た のに 、 すごい 眼 が いい じゃ ない か 」
「 俺 は おかしな 年寄り だ から な 」
「 超 大物 と も 戦える ？ 」
「 大丈夫 だ 。 やり方 は 色々 と ある 」
「 道具 を 片付ける う か 」 少年 は 言う た 。 「 それから 、 投網 を 持つ て イワシ を 獲る り に 行く よ 」
　 二 人 は 船 から 道具 を 取り出す た 。 老人 は マスト を 肩 に かつぐ 、 少年 は 木 箱 と 手鉤 《 ギャフ 》 と 柄 《 え 》 つき の 銛 《 もる 》 を 運ぶ だ 。 木 箱 に は 、 撚る 《 よ 》 り の 強い 茶色 の ロープ が 渦 に なる て 収まる て いる た 。 餌 に する 魚 を 入れる た 箱 は 、 棍棒 と 一緒 に 船尾 の ほう に 残す て おく た 。 棍棒 は 、 大きな 魚 を 船 《 ふる な 》 べり まで 引き寄せる た 時 、 魚 が 暴れる の を 鎮める ため に 使う もの だ 。
　 老人 から 何 か 盗む う と する 者 など いる ない だ う けれど 、 帆 や 重い ロープ に は 夜露 は よい ない から 、 持ち帰る た ほう が 良い 。 老人 も 、 この 辺 の 人間 が 自分 の 物 を 盗む こと など ない と 信じる て は いる た が 、 手鉤 《 ギャフ 》 や 銛 を 船 に 残す て おく と 人 の 出来心 を 不 必要 に 誘う かねる ない と も 思う て いる た 。
　 二 人 は 、 老人 の 棲家 だ ある 粗末 だ 小屋 まで 一緒 に 歩く 、 開け放す て ある 入り口 から 中 へ 入る た 。 老人 は 、 帆 を 巻く つける た マスト を 壁 に 立てかける 、 少年 は その そば に 木 箱 や 他 の 道具 を 置く た 。 マスト は 、 小屋 に 一つ しか ない 部屋 の 、 奥行き と 同じ くらい 長い た 。 小屋 は 、 グアノ と 呼ぶ れる ダイオウ ヤシ の 、 若芽 を 包む 硬い 苞 《 ほう 》 で できる て いる た 。 中 に は 、 ベッド 、 テーブル 、 椅子 が 一 脚 ある 、 土間 に は 炭 を 使う て 炊事 が 出来る 場所 も ある 。 丈夫 だ 繊維 から なる グアノ の 葉 を 伸ばす て 重ねる て 作る れる た 褐色 の 壁 に は 、 多色 刷り の 絵画 が 二 枚 掛ける られる て いる た 。 『 イエス の 聖心 』 、 そして 『 コブ レ の 聖母 マリア 』 。 どちら も 妻 の 形見 だ た 。 かつて は 、 色あせる た 妻 の 写真 も その 壁 に 飾る れる て いる た が 、 見る たび に 老人 は あまりに も 淋しい 気持ち に なる た ので 、 取り外す て しまう た 。 今 で は その 写真 は 、 部屋 の 隅 に ある 棚 の 、 洗濯 する た シャツ の 下 に しまう て ある 。
「 何 を 食べる の ？ 」 少年 が 尋ねる た 。
「 魚 の 混ぜる 飯 《 めし 》 が ある 。 食べる か ？ 」
「 僕 は 家 で 食べる よ 。 火 を 起こす う か ？ 」
「 いや 、 後で 自分 で 起こす う 。 冷たい まま で 食べる て も いい 」
「 投網 は 持つ て 行く て いい ？ 」
「 ああ 」
　 網 など 無い た 。 網 を 売る て しまう た 時 の こと は 、 少年 も 覚える て いる 。 しかし 二 人 は この 虚構 を 毎日 繰り返す て いる た 。 魚 を 混ぜる た 飯 も 無い 。 少年 は それ も 知る て いる 。
「 八 五 って の は 、 良い 数字 だ 」 老人 は 言う た 。 「 バラ する て 千 ポンド に も なる よう だ 、 大物 を 釣る て くる の を 見る たい だ う ？ 」
「 僕 、 投網 を 持つ て 行く て 、 イワシ を 取る て くる よ 。 戸口 の 、 日 の 当たる 所 で 座る てる て くれる ？ 」
「 ああ 。 昨日 の 新聞 が ある から 、 野球 の 記事 でも 読む で いる う 」 昨日 の 新聞 という の も 作り話 だ の か どう か 、 少年 に は 分かる ない た 。 しかし 老人 は ベッド の 下 から 新聞 を 取り出す た 。
「 酒屋 《 ボデガ 》 で ペリコ が くれる た ん だ 」 老人 は そう 説明 する た 。
「 イワシ が 獲る れる た 戻る て くる よ 。 サンチャゴ の 分 と 自分 の 分 と 一緒 に 、 氷 に 乗せる て おく 。 明日 の 朝 に 分ける う 。 戻る て くる た 、 野球 の 話 を 聞く せる て よ 」
「 勝つ の は ヤンキース だ う 」
「 でも クリーブランド・インディアンス が 怖い な 」
「 お前 、 ヤンキース を 信じる ん だ よ 。 大 ディマジオ が いる じゃ ない か 」
「 デトロイト ・ タイガース も 、 クリーブランド・インディアンス も 強い から なあ 」
「 しっかり する よ 、 その 調子 だ と シンシナティ ・ レッズ とか シカゴ ・ ホワイト ソックス まで 怖い なる ぞ 」
「 とにかく その 辺 を 読む で おく て よ 、 戻る て くる た 聞く から ね 」
「 下 二 ケタ が 八 五 の くじ を 買う て おく という の は どう だ 。 明日 は 八 五 日 目 だ から な 」
「 いい ね 」 少年 は 言う た 。 「 でも 、 八 七 の ほう が いい ん じゃ ない ？ 　 サンチャゴ の すごい 記録 じゃ ない か 」
「 あんな こと は 二度と 起きる ない だ う 。 八 五 の くじ を 探せる か ？ 」
「 買える よ 」

「 頑張る て ね 、 サンチャゴ 」

　 時々 、 どこ か の 船 から 話し声 が する こと も ある た が 、 ほとんど の 船 は 静か だ 、 ただ オール の 音 だけ が 聞こえる て くる た 。 港 の 出口 を 越える と 、 船 たち は ばらばら に 拡がる て いく た 。 それぞれ が 魚 の 居場所 に あたり を つける て 大海 へ 向かう の だ 。 老人 は 遠い まで 行く つもり だ た 。 彼 は 陸 の 匂い を 後 に する て 、 朝 の 海 の 清々しい 匂い の 中 を 漕ぐ で 行く た 。 水 の 中 で 、 ホンダワラ と おぼしい 海藻 が 燐光 《 りん こう 》 を 放つ て いる の が 見える た 。 深い さ が 突然 七 〇 〇 ｜ 尋 《 ひろ 》 に も なる て いる この 辺り を 、 漁師 たち は 「 大井戸 」 と 呼ぶ 。 海底 の 急斜面 に 海流 が ぶつかる て 渦 を 作る 、 あらゆる 種類 の 魚 が 集まる 場所 だ た 。 小エビ や 小 魚 《 ベイトフィッシュ 》 、 時には 深い 穴 の 中 に ヤリイカ の 群れ が いる こと も ある 。 それら が 夜 に なる て 海面 近い まで 上がる て くる と 、 泳ぎ 回る 魚 たち の 格好 の 餌 に なる の だ 。
　 暗闇 の 中 で 、 老人 は 夜明け が 近い の を 感じる た 。 漕ぐ ながら 聞こえる て くる の は 、 海面 から 跳ねる 出る トビウオ が 身 を 震わせる 音 や 、 闇 の 中 を 飛ぶ 去る 彼ら が その 硬い 頑丈 だ 翼 で 風 を 切る 音 だ た 。 老人 は トビウオ を 愛す た 。 海 の 上 で は トビウオ が 一番 の 友 だ た から だ 。 そして 彼 は 鳥 を 哀れむ だ 。 特に 、 小さい て か弱い 、 黒い アジ サシ 。 いつ でも 飛び回る て 餌 を 探す て いる のに 、 ほとんど 何 も 見つける られる ない 。
　 老人 は 考える 。 鳥 の 一生 は 、 俺 たち の 人生 より 苦しい 。 泥棒 鳥 とか 、 でかい て 強い 鳥 は 別 だ が 。 なぜ ウミツバメ みたい だ 、 弱い て 繊細 だ 鳥 が つくる れる た ん だ う 、 この 残酷 だ 海 に 。 いや 、 海 は 、 優しい て 美しい 。 でも 残酷 だ 。 突然 残酷 に なる ん だ 。 悲しい げに 小さな 声 で 鳴く ながら 飛び回る 、 急降下 する て 餌 を 取る う と する あの 鳥 たち は 、 この 海 で 生きる に は あまりに 弱い 。
　 彼 にとって 海 は 「 ラ・マール 」 だ ある た 。 海 を 愛する 人々 は 、 海 の こと を スペイン 語 の 女性 形 で そう 呼ぶ 。 時に 海 を 悪い 言う 場合 でも 、 彼ら にとって 海 は 女性 だ の だ た 。 若い 漁師 の 中 に は 、 ロープ に 繋ぐ 浮き として ブイ を 使う たり 、 サメ の 肝臓 で 儲ける た 金 で モーターボート を 買う たり する 者 が いる て 、 そういう 者 は 海 を 「 エル ・ マール 」 と 男性 形 で 呼ぶ で いる た 。 そういう 若者 にとって 、 海 は ライバル だ ある たり 単なる 場所 だ ある たり 、 場合 によって は 敵 で さえ ある た 。 しかし 老人 にとって 海 は いつも 女性 だ ある 、 大きな 恵み を くれる たり くれる ない たり する もの だ た 。 野蛮 だ こと や 危険 だ こと も する が 、 それ は 彼女 自身 どうにも 止める られる ない こと だ 。 女性 に 月 が 影響 する の と 同じ だ 、 海 に も 月 が 影響 する 。 老人 は そう 考える て いる た 。
　 老人 は 弛い 《 たゆい 》 みる ない 漕ぐ だ 。 自分 の ペース を 保つ て いる 分 に は 、 それほど 力 を 込める 必要 も ない 。 時に 潮 の 流れ が 渦巻く て いる の を 除く ば 、 海面 は 静か だ た 。 老人 は 船 を 動かす 仕事 の 三 分の 一 を 流れ に 任せる た 。 明るい なる 始める た 頃 に は 、 予定 より ずっと 遠い まで 来る て いる た 。
　 この 一 週間 は 大井戸 で 粘る た が 、 収穫 は ない た 。 今日 は カツオ や ビンナガ マグロ の 群れ が いる 辺り を 狙う て やる う 、 その 群れ の 中 に でかい 奴 が いる かも しれる ん 。 彼 は そう 考える て いる た 。
　 夜 が 明ける きる 前 に 老人 は 仕掛け を 下ろす 、 船 の 動き を 流れ に 預ける た 。 一つ 目 の 仕掛け は 四 〇 尋 まで 沈める た 。 二つ 目 は 七 五 尋 、 三つ 目 と 四つ 目 は さらに 海中 深い 、 一 〇 〇 尋 と 一 二 五 尋 まで ロープ が 届く て いる 。 それぞれ の ロープ の 先 で は 、 餌 と なる 小 魚 が 頭 を 下 に する て 鉤 《 かぎ 》 の 軸 に 体 を 貫く れる 、 きつい 縫い 刺 《 ざ 》 する に する れる て いる た 。 鉤 の 先 の 、 小 魚 の 体 から 突き出る た 部分 は 、 曲がる て いる 部分 も 先 っぽい も 、 新鮮 だ イワシ で 覆う れる て いる た 。 イワシ たち は 眼 を 串刺し に する れる 、 突き出る た 鋼鉄 の 棒 の 先 に 咲く た 半円 形 の 花びら の よう だ た 。 大きな 魚 が どこ から 近づく て も 、 すばらしい 匂い と 味 を 感じる られる はず だ 。
　 少年 が くれる た 新鮮 だ 餌 は 、 小さな マグロ 二 匹 だ た 。 ビンナガ という やつ だ 。 その 二 匹 は 、 深い ほう の 二 本 の 仕掛け に おもり の よう に 吊る 下げる られる て いる た 。 別 の 二 本 に は 、 大きな ヒラアジ と コガネ アジ が 付く て いる 。 こちら は 昨日 も 使う た 餌 だ が 、 まだ 十 分 使える 状態 だ 。 そこ に 、 匂い で 獲物 を 惹く つける ため の 新鮮 だ イワシ も 一緒 に 付ける て ある の だ た 。
　 四 本 の ロープ は どれ も 太め の 鉛筆 ほど の 直径 で 、 切る た ばかり の 生木 《 だ まき 》 の 枝 に 結ぶ れる て いる た 。 魚 が 餌 に 触れる たり 餌 を 引っ張る たり する ば 、 枝 が たわむ で 合図 と なる 仕組み だ 。 どの ロープ に も 、 四 〇 尋 の ロープ を 二 本 ずつ 、 控え として 付ける て ある 。 控え の ロープ 同士 は 繋げる られる よう に なる て いる ので 、 いざ という 時 に は 、 三 〇 〇 尋 以上 の 一 本 の ロープ に する て 魚 に 対応 できる の だ た 。
　 老人 は 今 、 船 べり を 超える て 突き出る た 三 本 の 枝 を 見つめる て いる 。 ロープ を 垂直 だ 状態 に する て 、 仕掛け の 深い さ が 変わる ない よう に 、 ゆっくり と 漕ぐ だ 。 辺り は もう 明るい 。 間もなく 日の出 だ う 。
　 海 から 、 うっすら と 太陽 が 昇る 始める 。 海流 の 向こう 側 、 ずっと 岸 寄り の ほう に は 、 他 の 船 たち が 海面 を 這う よう に 散らばる て いる の が 見える た 。 太陽 が 輝く 、 海面 を ぎらぎら と 照らす と 、 その 光 は 平ら だ 海 に 反射 する て 、 老人 の 眼 を 鋭い 突き刺す 。 太陽 は もう すっかり 姿 を 見せる て いる 。 老人 は 顔 を 背ける て 漕ぐ だ 。 そして 水 を 覗き 込む 、 暗い 海 の 中 へ と まっすぐ に 垂れる た ロープ を 見つめる た 。 彼 は 水中 の ロープ を 垂直 に 保つ て おく の が 誰 より 上手い た 。 この 技術 によって 老人 は 、 全て の 餌 を 望む 通り の 深い さ に 正確 に 配置 する 、 そこ を 泳ぐ 魚 を 狙う こと が できる の だ た 。 他 の 漁師 たち は 餌 が 流れ に 漂う こと を 気 に する ない から 、 一 〇 〇 尋 の 深い さ を 狙う て いる つもり が 実際 の 餌 は 六 〇 尋 の 位置 に ある たり する 。
　 だが 俺 の 腕 は 確か だ 、 と 老人 は 考える た 。 ただ 運 に 見放す れる てる だけ だ 。 いや 、 そう と も 限る ん 、 今日 は きっと いける 。 毎日 が 、 新しい 一 日 だ 。 運 は ある た ほう が いい が 、 運 任せる で は 駄目 だ 。 そういう 気持ち で いる ば 、 運 が めぐる て くる た 時 に 慌てる こと も ない 。
　 日の出 から 二 時間 が 経つ た 。 東 の ほう を 見る て も 、 眼 は さほど 痛む ない 。 視界 に 入る 船 は 三つ だけ だ た 。 どれ も 遠い 岸 寄り に いる て 、 海面 に 貼る 付く て 見える 。
　 俺 の 眼 は 、 明け方 の 太陽 に ずっと 痛めつける られる て くる た 。 だ が 今 でも この 眼 は よく 見える 。 夕方 に なる ば 、 太陽 を 直視 する て も 眼 が くらむ こと は ない 。 夕方 の ほう が 光 は 強い くらい だ が な 。 それにしても 、 朝 の 光 という の は きつい もの だ 。
　 ちょうど その 時 、 前方 の 空 に 、 黒い 長い 翼 を 持つ 軍艦 鳥 《 ぐんかんどり 》 が 旋回 する の が 見える た 。 軍艦 鳥 は 、 翼 を 後ろ に そらせる て 、 斜め に 急 降下 する た 。 そして また 旋回 を 始める た 。
「 何 か 見つける やる がる た な 」 老人 は 声 に 出す て 言う た 。 「 ただ 探す てる 時 の 飛ぶ 方 じゃ ない 」
　 老人 は 、 鳥 が 旋回 する あたり に 向かう て 、 安定 する た 動き で ゆっくり と 漕ぐ で 行く た 。 決して 急ぐ ぬ 、 ロープ は 垂直 に 保つ た まま だ 。 正しい 釣る 方 を 崩す ぬ に 、 と は いう 鳥 を 目標 として 多少 は 速度 を 上げる ため に 、 老人 は 少し だけ 潮流 に 近づく て 進む で 行く た 。

「 シイラ だ 」 老人 は 言う た 。 「 でかい シイラ が いる 」

　 陸地 の ほう に は 、 雲 が 山 の よう に 盛り上がる て いる た 。 海岸 は 、 長い 続く 一 本 の 緑色 の 線 で しか ない 、 その 背景 に は くすむ だ 青色 の 丘 が 並ぶ で いる 。 海 の 青 は 暗い 、 ほとんど 紫色 の よう だ た 。 海 の 中 を 見下ろす と 、 暗い 水中 に 赤い 散らばる プランクトン や 、 太陽 の 作り出す 不思議 だ 光 の 模様 が 見える た 。 老人 は 、 闇 の 中 に まっすぐ に 垂れ下がる ロープ を 見つめる 、 水中 に プランクトン が 多い こと を 喜ぶ だ 。 それ は 、 魚 が いる 印 だ から だ 。 高い 昇る た 太陽 が 、 水中 に あの 不思議 だ 光 の 模様 を 作る の は 、 天気 の いい 印 だ ある 。 陸地 の 上 の 雲 の 形 も そう だ 。

「 アグア・マーラ か 」 彼 は 言う た 。 「 娼婦 め 」

　 虹 色 の 泡 は 美しい 。 しかし 奴ら は 海 で 一番 の 詐欺 師 だ 。 老人 は 、 大きな 海 亀 が 奴ら を 食べる て しまう の を 見る の が 大好き だ た 。 海 亀 は 、 カツオノエボシ を 見つける と 、 正面 から 近づく て 行く 、 眼 を 閉じる て 全身 の 守り を 固める 、 触手 ごと 丸々 食べる て しまう の だ 。 老人 は 、 海 亀 が 奴ら を 食べる 様子 を 見る の も 、 嵐 の 後 の 浜辺 で カツオノエボシ の 上 を 歩く て 、 硬い かかと で 踏みつける て ポ ン と 破裂 する 音 を 聞く の も 好き だ た 。
　 彼 は 、 アオウミガメ や タイマイ を 愛す て いる た 。 優雅 だ 、 泳ぎ が 速い 、 値打ち が ある から だ 。 大きい て 愚か だ アカ ウミガメ に は 、 軽蔑 混じり の 親しみ を 抱く て いる た 。 黄色い 鎧 を 着ける て 、 おかしな 求愛 行動 を する 奴 で 、 カツオノエボシ を 食べる 時 に は 幸せ そう に 眼 を 閉じる の だ た 。
　 老人 は 過去 に 何 年 も 海 亀 獲る り の 船 に 乗る て いる た が 、 海 亀 を 神秘 的 だ 生き物 だ と は 思う て いる ない た 。 むしろ 哀れ に 思う て いる た 。 小船 と 同じ くらい の 大きい さ で 体重 は 一 トン も ある う か という 巨大 だ オサガメ の こと さえ 、 哀れむ で いる た 。 ほとんど の 人間 は 、 海 亀 に対して 冷淡 だ 。 海 亀 の 心臓 は 、 体 が 切り刻む れる て も 一 時間 は 動く 続ける から だ 。 しかし 老人 は 、 自分 の 心臓 だって 同じ だ と 思う て いる た 。 脚 や 手 だって 、 亀 と 変わる ない 。 彼 は 力 を つける ため に 、 海 亀 の 白い 卵 を 食べる て いる た 。 五月 中 は 卵 を 毎日 食べる て 力 を つける 、 九月 から 十月 に は 超 大物 と 戦う の だ 。
　 彼 は サメ 肝油 も 飲む で いる た 。 漁師 たち の 道具 小屋 に ある 、 大きな ドラム缶 から 、 毎日 一 杯 ずつ 飲む 。 飲む たい 漁師 は 誰 でも 飲める よう に 置く て ある の だ が 、 たいてい の 漁師 は その 味 を 嫌う て いる た 。 しかし 、 早い 時間 に 起き出す 大変 さ に 比べる ば 何 て こと は ない し 、 飲む で いる ば 風邪 や インフルエンザ に も 強い なる 。 おまけ に 眼 に も 良い の だ 。
　 老人 が ふと 見上げる と 、 あの 鳥 が 、 また 旋回 を 始める て いる た 。
「 魚 を 見つける た な 」 彼 は 声 に 出す て 言う た 。 海面 を 跳ねる トビウオ は おる ぬ 、 小 魚 が 散らばる 様子 も ない た 。 が 、 老人 が 見る て いる と 、 小さな マグロ が 一 匹 跳ね上がる 、 空中 で 逆さ に なる て 頭 から また 水 に 潜る た 。 日光 で 銀色 に 輝く その マグロ が 水中 に 消える て しまう と 、 次 から 次 へ と マグロ たち が 飛び上がる 、 四方八方 に 跳ねる まくる た 。 水 を かき回す 、 餌 を 求める て 大きい 飛び跳ねる 。 そして 輪 を 描く て 獲物 を 追い込む う と する て いる た 。
　 マグロ たち の 動き が あんなに 速い ない ば 、 あの 真ん中 に 船 を 突っ込む で やる ん だ が な 。 老人 は そう 考える た 。 マグロ の 群れ が 水 を 白い 泡立つ せる 様子 や 、 パニック に なる て 水面 に 上がる て くる た 小 魚 を 狙う て 鳥 が 急 降下 を 繰り返す 様子 を 、 老人 は 見つめる て いる た 。
「 あの 鳥 に は ずいぶん 世話 に なる 」 老人 は 言う た 。 その 時 、 彼 の 足 の 下 で 輪 に する て ある た ロープ が 、 船尾 の ほう から ぐっと 引っ張る れる た 。 老人 は オール から 手 を 離す た 。 ロープ を 堅い 握る て 手繰る 寄せる と 、 小さな マグロ が 体 を 震わせる ながら 引っ張る て いる 力 を 感じる 。 震える 力 は 、 ロープ を 引く ば 引く ほど 強い なる て いく た 。 水 の 中 から 、 魚 の 青い 背中 と 金色 の わき腹 が 見える て くる た 。 そして 彼 は 、 船 べり を 越える て 魚 を 船 に 引き入れる た 。 魚 は 太陽 に 照らす れる 、 引き締まる た 弾丸 の よう だ 体 を する て 、 大きい 無表情 だ 目 を 見開く ながら 、 良い 形 の よく 動く 尾 を 素早い 震わせる 、 船板 《 ふる ない た 》 に その 生命 を 打ちつける て いる 。 老人 は 優しい さ から 、 魚 の 頭 を 叩く 、 蹴飛ばす た 。 魚 は 船尾 の 陰 に 飛ぶ 、 それでも 震える 続ける て いる 。
「 ビンナガ だ 」 老人 は 声 に 出す て 言う た 。 「 いい 餌 に なる ぞ 。 十 ポンド は ある そう だ 」
　 声 に 出す て 独り言 を 言う よう に なる た の は いつ から だ た か 、 彼 は 覚える て いる ない た 。 昔 は 、 一 人 の とき に は 歌 を 口ずさむ だ もの だ 。 魚 槽 付 小型 船 《 スマック 》 や 亀 獲る り 船 で 、 寝る ぬ に 舵取り の 番 を する よう だ 時 など 、 たまに 歌 を 唄う て いる た 。 老人 が 独り言 を 言う よう に なる た の は きっと 、 少年 が 去る て 、 一 人 に なる て から だ う 。 しかし 定か だ は ない た 。 少年 と 一緒 に 漁 に 出る て いる た 頃 は 、 必要 だ 時 以外 ほとんど 会話 を する ない た 。 二 人 が 話す の は 、 夜 とか 、 悪天候 で 船 を 出せる ない とき だ 。 海 で は 不 必要 に 喋る ない の が 美徳 だ た し 、 老人 は いつ でも その 美徳 を 尊重 する て いる た 。 しかし 今 、 彼 は 自分 が 思う た こと を たびたび 声 に 出す 。 それ で 困る 者 も いる ない から だ 。
「 べらべら 喋る て いる の を 誰 か が 聞く た 、 俺 の こと を 気違い だ と 思う だ う な 」 老人 は 声 に 出す て 言う た 。 「 だが 気違い じゃ ない 。 だから 構う ない ん だ 。 金持ち の 奴ら など ラジオ を 持つ て いる て 、 船 の 中 で 喋る せる どころか 、 野球 の 実況 まで する せる てる じゃ ない か 」
　 今 は 野球 の こと を 考える 時 じゃ ない 、 彼 は 思う た 。 考える べし は 、 たった 一つ 。 その たった 一つ の ため に 、 俺 は 生まれる て くる た の だ 。 この 群れ の まわり に 、 大物 が いる かも しれる ない 。 まだ 、 餌 を 追う ビンナガ の 群れ から 逸 《 はぐ 》 れる た 一 匹 を 釣る た だけ だ 。 しかし 群れ は もう 遠い 、 速い 。 今日 は 水面 に 見える 全て が 、 北東 に 向かう て 高速 で 動く て いる よう だ 。 時間 帯 の せい か 。 それとも 、 俺 の 知る ない 、 天気 が 変わる 前兆 だ う か 。
　 緑色 の 海岸 線 は もう 見える ない 。 見える の は 、 青い 丘 の 頂 《 いただき 》 が まるで 雪 を かぶる た か の よう に 白い 光る 様子 と 、 その 上 に 高い 雪山 の よう に 広がる 雲 だけ だ た 。 海 は とても 暗い 、 差し込む 光 が 水 の 中 に プリズム を 作る て いる た 。 無数 の 斑点 の よう だ プランクトン たち の 姿 は 、 高い 昇る た 日 の 光 で かき消す れる て いる 。 老人 の 目 に 映る の は 、 青い 水 の 中 に 深々 と 延びる プリズム と 、 一 マイル の 深い さ まで 真っ直ぐ に 垂らす れる て いる 彼 の ロープ だけ だ た 。
　 マグロ たち は —— 漁師 は この 種 の 魚 を 全て マグロ と 呼ぶ 、 売る たり 餌 と 交換 する たり する 時 だけ それぞれ の 名前 を 用いる て 区別 する て いる た ——、 再び 潜る て しまう た 。 太陽 は もう 熱い 、 老人 は うなじ で それ を 感じる た 。 漕ぐ ながら 、 背中 に 汗 が 流れる の が 分かる た 。

「 食いつく ぞ 」 老人 は 言う た 。 「 どう か 、 食いつく て くれる 」

やがて 優しい げ だ 引き を 感じる 、 老人 は 喜ぶ だ 。
「 ひと 回り する て いる た だけ だ 」 彼 は 言う た 。 「 食いつく ぞ 」

　 彼 は 実行 する た 。 闇 の 中 で は 難しい 作業 だ た 。 一度 、 魚 が うねる よう に 大きい 動く 、 彼 は 顔 から 引く 倒す れる て 目の下 を 切る た 。 少し だけ 頬 に 血 が 流れる た が 、 顎 に 届く 前 に 固まる て 乾く て しまう た 。 何とか 舳先 《 へ さき 》 まで 戻る 、 船板 に も たれる て 休む だ 。 彼 は 、 肩 に 当てる た 袋 の 位置 を 調整 する て 、 ロープ の 当たる 場所 を 慎重 に 変える た 。 肩 で 支える ロープ から 伝わる 引き 具合 を 注意深い 確かめる つつ 、 片手 を 水 に 入れる て 船 の 進む 速い さ を 測る た 。
　 なぜ 奴 は 、 急 に あんなに ふらつく た ん だ う 。 彼 は 思う た 。 きっと 、 大きい 盛り上がる た 背中 に 針金 が こすれる た ん だ 。 俺 の 背中 ほど ひどい 痛み で は ない だ う が な 。 しかし 奴 が いくら 立派 だ と 言う て も 、 この 船 を 永遠 に 引き続ける わけ に は いく まい 。 もう 心配 だ こと は 何 も ない し 、 控え の ロープ も 十分 に ある 。 やれる こと は 全て やる た 。
「 魚 よ 」 彼 は 声 に 出す て 、 優しい 言う た 。 「 俺 は 死ぬ まで お前 と 一緒 だ 」
　 奴 も きっと その つもり だ う 、 老人 は そう 思う て 、 夜明け を 待つ た 。 日の出 前 の この 時間 は 冷える 。 彼 は 船板 に 体 を 押し付ける て 寒い さ を しのぐ だ 。 奴 が やる 限り は 、 俺 も やる て やる 。 彼 は そう 考える た 。 明け方 の 薄 明かり の 中 で 、 ロープ は まっすぐ 水中 へ と 走る て いる 。 船 は 着実 に 移動 を 続ける 。 やがて 太陽 の ふち が 水平 線 から 現れる 、 老人 の 右肩 を 照らす た 。
「 北 に 向かう てる 」 老人 は 言う た 。 しかし 潮 の 流れ は 、 俺 たち を ずっと 東 まで 運ぶ だ う な 。 奴 が 流れ に 押す れる て 向き を 変える て くれる よう だ と いい ん だ が 。 彼 は そう 考える た 。 そう なる ば 、 魚 が 疲れる て いる こと が 分かる 。
　 太陽 が さらに 昇る 、 老人 は 、 魚 は 疲れる て など いる ない と 知る た 。 一つ だけ 、 好ましい 徴候 が ある た 。 魚 の 泳ぐ 水深 が やや 浅い なる て いる こと を 、 ロープ の 角度 が 示す て いる た の だ 。 だから と 言う て 必ず 跳ね上がる と は 限る ない 。 が 、 跳ね上がる かも しれる ない 。
「 どう か 、 跳ねる て くれる 」 老人 は 言う た 。 「 奴 を 扱える だけ の ロープ は 用意 する て ある 」
　 おそらく 、 俺 が ほんの 少し ロープ の 張り を 強める だけ だ 、 奴 は 痛い がる て 跳ね上がる だ う 。 彼 は 考える た 。 もう 夜 は 明ける た ん だ 、 奴 を 跳ねる させる て やる う 。 そう する て 奴 の 背骨 近く の 浮き袋 が 空気 で 膨れる て しまう ば 、 もう 深い 潜る て 死ぬ こと は できる ない なる 。
　 彼 は 張り を 強める う と する た が 、 ロープ は 既に 、 ちぎれる 寸前 まで 張り切る て いる た 。 あの 魚 が 掛かる て から ずっと そう だ た の だ 。 後ろ へ 体 を 傾ける て 引く と 荒々しい 手応え が ある 、 これ 以上 引っ張る こと は できる そう も ない 。 引く ちゃ いかん 、 と 彼 は 思う た 。 もし 引く ば 、 奴 の 傷 が 広がる て 、 跳ね上がる た 時 に 鉤 を 振り落とす れる かねる ない 。 ともかく 、 日 が 昇る て から 俺 の 調子 も 良い なる た 。 それ に 、 もう 太陽 を 直視 する ない て 済む 。
　 ロープ に は 黄色い 海藻 が まとわりつく て いる た 。 魚 にとって は 重り が 増える だけ だ から 、 老人 は むしろ 喜ぶ だ 。 この 黄色い ホンダワラ が 、 昨晩 あれ ほど 燐光 を 放つ て いる た の だ 。
「 魚 よ 」 彼 は 言う た 。 「 お前 を 愛す てる し 、 心から 尊敬 する て も いる 。 だが 、 俺 は お前 を 必ず 殺す 。 今日 の うち に な 」
　 そう なる て 欲しい もの だ 、 と 彼 は 思う た 。
　 小さな 鳥 が 、 船 に 向かう て 北 の ほう から 飛ぶ で くる た 。 鳴る 鳥 の 一種 で 、 海上 を かなり 低い 飛ぶ で いる 。 老人 に は 、 鳥 が ずいぶん 疲れる て いる こと が 分かる た 。
　 小鳥 は 船尾 の ほう に 来る て 、 そこ に とまる た 。 それから 老人 の 頭 の まわり を 飛ぶ 、 今度 は もっと 居心地 の 良い さ そう だ 、 ロープ の 上 に とまる た 。
「 何 歳 《 いくつ 》 だい ？ 」 老人 は 鳥 に 尋ねる た 。 「 旅 は 初めて か ？ 」
　 喋る 老人 を 、 鳥 は 見つめる て いる た 。 疲れ 果てる て 足場 を あらためる 余裕 も ない 鳥 は 、 華奢 だ 足 で しっかり ロープ を 握る て 、 その 上 で 揺る れる て いる 。
「 その 綱 は 丈夫 だ 」 老人 は 鳥 に 言う た 。 「 かなり 丈夫 だ ぞ 。 昨晩 は 風 も 無い た のに 、 そんなに 疲れる て いる た いく ん な 。 この 先 どう する つもり だ ？ 」
　 彼 は 、 小鳥 たち を 狙う て 海 に まで 来る 鷹 の こと を 考える た 。 しかし 何 も 言う ない た 。 言う て も 鳥 に は 理解 できる ない し 、 言う ない て も すぐ に 分かる こと だ 。
「 小鳥 よ 、 しっかり 休む で 行く よ 」 彼 は 言う た 。 「 それから 陸 の ほう へ 飛ぶ ん だ 。 後 は 運 任せる さ 、 人間 も 鳥 も 魚 も 同じ だ う ？ 」
　 会話 は 老人 を 元気づける た 。 彼 の 背中 は 夜 の うち に 硬い なる て 、 いまや ひどい 痛む で いる た 。
「 よい た 、 家 《 うち 》 に 泊まる と いう 」 彼 は 言う た 。 「 しかし 残念 だ が 、 帆 を 上げる て 陸 まで 連れる て く て やる という わけ に は いく ん 。 ちょうど 風 も 出る て くる た ところ だ が な 。 いかんせん 、 俺 に は 連れ が いる ん だ 」
　 その 時 、 魚 が 突然 ふらつく て 、 老人 は 舳先 《 へ さき 》 に 引く 倒す れる た 。 ふんばる て ロープ を 送り出す た から よい た ものの 、 そう する ない ば 水 の 中 に 引きずり込む れる て いる た だ う 。
　 ロープ が 引っ張る れる た ので 小鳥 は 飛び立つ た 。 小鳥 が 去る の を 見る て いる 余裕 は 老人 に は 無い た 。 彼 は 右手 で 慎重 に ロープ を 握る 、 自分 の 手 から 血 が 流れる て いる こと に 気付く た 。
「 奴 は 、 どこ か 痛い た ん だ う な 」 彼 は 声 に 出す て 言う た 。 そして 、 魚 の 向き を 変える られる ない か と ロープ を 引っ張る て みる た 。 しかし 、 切れる 寸前 まで 引く と 、 そこで 張り を 安定 する せる 、 後ろ に 体重 を かける て ロープ を 支える た 。
「 魚 よ 、 そろそろ こたえる て くる た だ う 」 彼 は 言う た 。 「 本当 の ところ 、 俺 も 同じ だ 」
　 彼 は あたり を 見回す て 鳥 を 探す た 。 仲間 が 欲しい た から だ 。 鳥 は もう いる ない た 。
　 長居 は する ない た ん だ な 、 老人 は 思う た 。 しかし 岸 まで の 道のり は もっと 辛い ぞ 。 魚 に 引っ張る れる て 手 を 切る なんて 、 俺 は 何 を やる てる ん だ 。 ずいぶん 馬鹿 に なる た もん だ 。 いや 、 きっと 小鳥 の こと なんか 考える て いる た せい だ な 。 自分 の 仕事 に 集中 する う 。 力 が 抜ける て しまう ない よう に 、 マグロ を 食う ない ば 。
「 あの 子 が ここ に いる て くれる た 。 それ に 、 塩 が ある た なあ 」 彼 は 声 に 出す て そう 言う た 。
　 ロープ の 重み を 左肩 に 掛ける かえる 、 慎重 に 膝 を つく て 、 彼 は 海水 で 手 を 洗う た 。 そして しばらく 海 の 中 に 手 を 浸す て 、 自分 の 血 が 水中 に 尾 を 引く 様子 を 見る ながら 、 船 の 動き が 生む 水 の 抵抗 を その 手 に 感じる 続ける て いる た 。
「 ずいぶん 速度 が 落ちる た な 」 彼 は 言う た 。
　 もう少し 手 を 塩水 に 浸す て おく たい と 思う た が 、 老人 は 、 また 魚 が 突然 ふらつく たり する こと を 心配 する た 。 彼 は 立ち上がる て 気 を 引き締める 直す 、 手 を 太陽 に かざす た 。 ロープ に こすれる て 皮膚 が 切れる た に すぎる ない 。 しかし 、 大事 だ 部分 だ 。 仕事 を 終える に は 両手 が 必要 だ の だ から 、 始める 前 から 怪我 を する こと は 避ける たい た 。
「 さあ 」 手 が 乾く と 彼 は 言う た 。 「 小さい マグロ を 食う ない いく ん 。 手鉤 《 ギャフ 》 を 使う ば 届く な 。 この まま ここ で 食う う 」
　 彼 は 膝 を つく 、 船尾 の マグロ の ほう に 手鉤 《 ギャフ 》 を 伸ばす た 。 そして 予備 の ロープ を 避ける つつ マグロ を 引き寄せる 。 再び ロープ を 左肩 で 支える 、 左 の 手 と 腕 に 力 を 入れる つつ 、 手鉤 《 ギャフ 》 の 先 から マグロ を 外す て 、 手鉤 《 ギャフ 》 は 元 の 場所 に 戻す た 。 片方 の 膝 で 魚 を 押さえる 。 そして 頭 の 後ろ から 尾 に 向かう て 縦 に 刃 を 入れる 、 赤黒い 魚肉 を 切り出す た 。 くさび 形 の 塊 だ 。 彼 は それ を 、 背骨 の そば から 腹 の 方向 へ 切る た 。 六つ の 切り身 が できる と 、 それ を 舳先 の 板 の 上 に 並べる 。 ナイフ を ズボン で 拭く 、 残る た 骨 の 尻尾 を つまむ で 海 に 投げ捨てる た 。
「 一 切れ 丸ごと は 食べる られる ん な 」 そう 言う て 彼 は 、 切り身 の 一つ に ナイフ を 入れる た 。 ロープ は 変わる ぬ 強い 引き続ける て いる 。 と 、 左手 が 引く つり を 起こす た 。 重い ロープ を きつい 握る た まま の 左手 を 、 彼 は うんざり する て 眺める た 。
「 なんて 手 だ 」 彼 は 言う た 。 「 つる たい つる が いい 。 鉤爪 《 かぎ づめ 》 に でも なる て しまう 。 何 の 役 に も 立つ ない ぞ 」
　 間抜け め 。 そう 思う ながら 彼 は 、 斜め に 走る ロープ の 先 の 暗い 海 を 見下ろす た 。 食う ない いかん 、 手 に 力 を つける ん だ 。 手 が 悪い わけ じゃ ない 。 もう 長い 時間 、 あの 魚 と こう する て いる ん だ から な 。 永遠 に でも 続ける て やる 。 さあ 、 マグロ を 食う ぬ ば 。
　 一 切れ を つまみあげる 、 口 に 入れる て 、 ゆっくり 噛む だ 。 まずい は ない 。
　 よく 噛む で 、 残らず 栄養 を 吸収 する ん だ 。 彼 は 考える た 。 ライム か レモン か 、 塩 で も ある ば いい ん だ が 。
「 具合 は どう だ ？ 」 彼 は 、 つる て いる 手 に 向かう て 尋ねる た 。 ほとんど 死後 硬直 の よう に 硬い なる て いる 。 「 お前 の ため に 、 もう少し 食う から な 」
　 彼 は 、 二つ に 切る た うち の 残り の 一 切れ を 口 に 入れる た 。 じっくり と 噛む で から 、 皮 を 吐き出す 。
「 さあ 、 どう だ ？ 　 そう 早い は 分かる ない か ？ 」
　 彼 は 次 の 切り身 を 取る 、 そのまま 口 に 入れる て 噛む だ 。
「 力 に 溢れる た 強い 魚 だ 」 彼 は 思う た 。 「 シイラ じゃ ない こいつ を 釣れる た の は 幸運 だ た な 。 シイラ で は 味 が いい だけ だ 。 こいつ は 旨み と は 無縁 だ が 、 力 が いっぱい に 詰まる てる 」
　 しかし 実際 の 味 を 全く 無視 する の も 無粋 という もの だ 、 と 彼 は 思う た 。 塩 が ある ば 良い た ん だ が 。 残す て おく と 、 日 に当たって 腐る か 干からびる か 分かる ない し 、 腹 は 減る て いる ない とも 全部 食べる て しまう た ほう が いい な 。 あの 魚 は 落ち着く て 安定 する てる 。 食べる て しまう て 、 後 に 備える う 。
「 手 よ 、 耐える 」 彼 は 言う た 。 「 お前 の ため に 食う ん だ から 」
　 あの 魚 に も 食う せる て やる たい 、 と 彼 は 思う た 。 奴 は 我が 兄弟 だ 。 しかし 殺す ない ば いける ない 。 その ため に は 強い いる ない ば 。 ゆっくり と 、 念入り に 噛む で 、 彼 は くさび 形 の 魚肉 を 全て 食べる 切る た 。
　 彼 は 立ち上がる 、 ズボン で 手 を 拭く た 。
「 さあ 」 彼 は 言う た 。 「 もう ロープ を 放す て しまう 。 お前 が 間抜け だ こと を やめる まで 、 俺 は 右手 だけ で 奴 を 制す て やる 」 彼 は 、 左手 が 握る て いる た 重い ロープ を 左足 で 押さえる 、 背中 に かかる 力 に 抵抗 する て 後方 に 体重 を かける た 。
「 引き つり が 治る こと を 祈る 」 彼 は 言う た 。 「 なにしろ 、 奴 が どう する つもり だ の か 分かる ない から な 」
　 それにしても 奴 は 落ち着く て いる 。 彼 は 考える た 。 計画 通り という こと か 。 だが 、 どんな 計画 だ う 、 と 彼 は 思う た 。 そして 俺 の 計画 は どう だ ん だ 。 俺 は 奴 に 合わせる て 即席 で やる しか ない 。 奴 は でかい から な 。 奴 が 跳ね上がる ば 、 殺す こと が できる 。 ところが 奴 は いつ まで も 深い 潜る て いる 。 それなら 、 俺 も いる つ まで も 付き合う う 。
　 彼 は 、 ひきつる 左手 を ズボン に こする つける 、 指 を 鎮める う と する た 。 しかし 手 は 開く ない 。 日 に 当たる て いる ば 開く だ う 、 彼 は そう 思う た 。 力 の ある 生 の マグロ が 消化 する れる た 開く かも しれる ない 。 開く 必要 が ある ば 、 何 が 何 でも 開く て やる さ 。 だが 、 いま 無理やり 開く う と は 思う ん 。 自然 に 開く て 、 ひとりでに 元 に 戻る の が 一番 だ 。 やはり 夜 の うち に 酷使 する すぎる た か 。 いくつ も ロープ を 切る たり ほどく たり する ない いける ない た から な 。

　 それから 長い 間 、 彼 は チャンピオン と 呼ぶ れる た 。 春 に は リターンマッチ も 行う れる た 。 しかし 賭け 金 は 少ない 、 彼 は その 金 を いとも 簡単 に 手 に する た 。 シエンフエゴス 出身 の 黒人 は 、 前 の 試合 で 自信 を 打ち砕く れる て いる た から だ 。 その後 、 彼 は 何 度 か 勝負 を する て 、 それ きり やめる て しまう た 。 本気 に なる ば どんな 相手 も 打つ 負かせる と 分かる た し 、 漁 に 使う 右手 に 腕相撲 を する せる の は 良い ない 。 何 度 か 、 試し に 左手 で やる て みる た こと も ある た 。 しかし 左手 は いつも 裏切り者 で 、 言う こと を きく ない 。 それで 彼 は 、 左手 を 信用 する ない なる た の だ た 。
　 太陽 が しっかり 手 を 温める て くれる だ う 。 彼 は 思う た 。 夜 に 極端 だ 冷え込み が 来る ない ば 、 引き つり は もう 無い はず だ 。 今夜 は どんな 夜 に なる の だ う か 。
　 飛行機 が マイアミ に 向ける て 頭上 を 飛ぶ で 行く た 。 その 影 に 驚く た トビウオ の 群れ が 、 跳ね上がる の が 見える た 。
「 これ だけ トビウオ が いる だ 、 シイラ も いる はず だ な 」 彼 は そう 言う ながら 、 あの 魚 を 多少 でも 引き寄せる こと が できる ない か 、 ロープ に 体重 を かける て みる た 。 しかし ロープ は 動く ぬ 、 切れる 寸前 で 硬い 張りきる て 、 水滴 を 散らす た 。 船 は ゆっくり と 進む 。 彼 は 、 飛行機 が 見える ない なる まで 眺める て いる た 。
　 飛行機 に 乗る た 、 きっと 妙 だ 気持ち に なる だ う な 、 と 彼 は 思う た 。 あの 高い さ から 、 海 は どう 見える の か 。 あまり 高い ところ を 飛ぶ ない ば 、 魚 が 見える だ う 。 二 百 尋 くらい の 高い さ を ゆっくり 飛ぶ で 、 上 から 魚 を 見る て みる たい 。 亀 獲る り 船 に 乗る て いる た 頃 、 マスト の 先 の 横木 《 クロスツリー 》 に よく 登る た もの だ が 、 その くらい の 高い さ で も ずいぶん 見える た 。 そこ から だ と シイラ は 濃い 緑色 に 見える て 、 縞 模様 と 紫色 の 斑点 も 見える ん だ 。 それ に 、 泳ぐ で いる 群れ の 全体 を 見渡せる 。 しかし なぜ 、 暗い 流れ の 中 を 素早い 泳ぐ 魚 という の は 、 どれ も みな 紫色 の 背中 で 、 紫色 の 縞 や 斑点 が ある ん だ う 。 シイラ は 本当は 金色 だ から 、 もちろん 緑色 に 見える ん だ が 、 ひどい 腹 を すく せる て 餌 を 食う 時 に なる と 、 側面 に カジキ みたい だ 紫色 の 縞 が 浮き出る ん だ 。 怒り の せい だ の か 、 それとも 、 普段 より スピード を 出す せい だ の か 。

　 もう 暗い た 。 九月 は 太陽 が 沈む と すぐ に 暗い なる 。 彼 は 舳先 の ぼろぼろ の 板 に も たれる て 、 できるだけ 楽 に 休む だ 。 空 に は 星 が 出る 始める て いる た 。 彼 は 星 の 名 を 知る ない た が 、 オリオン 座 の リゲル を 見つける 、 じきに 他 の 星 たち も みな 出る て くる だ う と 分かる た 。 星 々 は 、 遠い ところ に いる 友人 たち だ 。

「 奴 を 殺す に は 他 の 方法 は 無い た 」 老人 は 言う た 。 水 を 飲む で から 少し 調子 が 良い なる て いる た 。 気 を 失う 心配 は もう 無い し 、 頭 も はっきり する て いる 。 丸 の まま で 千 五 百 ポンド は ある な 、 と 彼 は 思う た 。 いや 、 もっと かも しれる ない 。 さばく て 三 分の 二 の 重い さ に なる として 、 一 ポンド あたり 三 十 セント だ いくら に なる だ う 。
「 鉛筆 が 要る な 」 彼 は 言う た 。 「 まだ 頭 が しっかり する てる ない 。 だ が 今日 の 俺 を 見る た 、 大 ディマジオ だって 褒める だ う 。 俺 に は 骨 棘 《 くる つ きょく 》 は 無い が 、 手 や 背中 は ひどい 傷 だ 」 骨 棘 と は どんな もの だ う な 、 と 彼 は 考える た 。 もしか する と 、 知る ない 間 に できる てる の かも しれる ない 。
　 魚 の ロープ は 船首 に も 船尾 に も 腰掛 梁 《 こしかける ばり 》 に も 結ぶ れる た 。 魚 は あまりに 大きい 、 この 船 より 大きな 船 を 横 に 括る つける た よう だ た 。 彼 は 短い 切る た ロープ で 魚 の 下 顎 と くちばし を 縛る た 。 口 を 閉じる させる て おく た ほう が 、 船 が 滑らか に 進む から だ 。 それから 彼 は マスト を 立てる 。 斜 桁 《 ガフ 》 と 下 桁 《 ブーム 》 の 間 に 継ぐ はぎの 帆 が 張る れる 、 船 は 動く 始める た 。 彼 は 船尾 で 半ば 寝そべる た まま 、 南西 へ 向かう た 。
